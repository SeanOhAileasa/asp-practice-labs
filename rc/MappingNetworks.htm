Exercise 1 - Working with Network Diagrams

Network mapping is the process of identifying or discovering the physical connectivity of computers in the network. You can perform network mapping with the help of Network Management Software (NMS). You can administer, monitor, manage, and scan the network using NMS. The visual network maps or diagrams provide help in the network mapping process. You can use the network topology mapper to plot the network automatically.

In this exercise, you will work with the Zenmap utility to obtain a graphic representation of the devices in the network. You will also work with ManageEngine OpManager application to demonstrate the mapping of the network.

Please refer to your course material or use your preferred search engine to research this topic in more detail.

Task 1 - Install Zenmap

In this task, you will install Zenmap. Netcat utility comes built-in with Zenmap GUI, which is the GUI version of Nmap. Netcat that comes prebuilt with Zenmap is called Ncat.

Step 1

Connect to PLABWIN10.

Access the Intranet Website by launching Internet Explorer.

Step 2

Navigate to Tools > Zenmap.

Step 3

Click on nmap-7.60-setup.exe.

When prompted, click Run.

Note: If you are prompted with the User Account Control dialog box, click Yes.

On the License Agreement page of the Nmap Setup wizard, read the license agreement and click I Agree.

Alert: Follow the installation steps through until completion. Keep all the default settings.

Step 4

On the Choose Components page, keep the default selection and click Next.

Step 5

On the Choose Install Location page, keep the default path and click Install.

Step 6

The Installing page is displayed. It shows the installation progress.

On the License Agreement page of the Npcap 0.93 Setup wizard, read the license agreement and click Next.

Step 7

On the Installation Options page, keep the default selection and click Install.

Step 8

On the Installation Complete page, click Next.

Step 9

Click Finish when the installation is complete.

Close Internet Explorer.

You are now back on the desktop. Note that the Nmap - Zenmap GUI icon is now placed on the desktop.

Task 2 - Identify Live Systems in the Network Using Zenmap

Zenmap is the official Nmap security scanner graphical user interface (GUI). In this task, you will use Zenmap to perform the following actions:

  • Identify live hosts in the network.
  • Display the identified hosts in a graphical presentation.
  • Set the configuration options, if required.

Step 1

Ensure you have powered on the required devices and connect to PLABWIN10.

From PLABWIN10 desktop, double-click Nmap - Zenmap GUI.The Zenmap window is now displayed.

Step 2

The Zenmap window opens.

To identify live hosts in the network, in the Command box, type the following command:

nmap -sn 192.168.0.1/24

Press Enter.

The Nmap scan report of the devices in the network is displayed.

Step 3

To display the identified hosts in GUI format, click the Topology tab.

Step 4

To display the identified network topology in Fisheye view, under the Topology tab, click the Fisheye tab.

Step 5

In the Fisheye tab, you can click and drag any host point to focus and change the view.

For the purpose of this demonstration, the host points are realigned.

Step 6

To set more configuration options, click the Controls tab.

The Controls pane appears.

Note: The Controls pane provides multiple control options such as Action, Interpolation, Layout, View, and so on.

You can set the desired options.
